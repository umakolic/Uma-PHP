# php-mysql-backend

**Ime i prezime studenta:** (upiši ovdje)  

**Naziv predmeta:** Softver inženjering  

**Opis projekta:** Web aplikacija za upravljanje korisnicima (CRUD) koristeći PHP (PDO) i MySQL, uz Bootstrap frontend.

## Funkcionalnosti

- Create (dodavanje korisnika)

- Read (prikaz liste korisnika + pretraga + paginacija)

- Update (izmjena korisnika)

- Delete (brisanje korisnika)
